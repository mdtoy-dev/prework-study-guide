HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
Text and Structure
  • h1, h2, ..., h6: Headings of decreasing importance.
  • p: Paragraph.
  • br: Line break.
  • hr: Horizontal rule (line).
  • em: Emphasis (italic).
  • strong: Strong importance (bold).
  • span: Generic container for inline elements.
Lists
  • ul: Unordered list.
  • ol: Ordered list.
  • li: List item.
Links
  • a: Anchor (creates a hyperlink).
Images
  • img: Image.
Forms
  • form: Form container.
  • input: Input field.
  • textarea: Text area.
  • button: Button.
  • select: Dropdown list.
  • label: Label for form elements.
Tables
  • table: Table container.
  • tr: Table row.
  • th: Table header cell.
  • td: Table data cell.
Semantic Elements
  • header, nav, main, article, section, aside, footer: Semantic elements for document structure.
Metadata
  • head: Container for metadata.
  • meta: Metadata tags (e.g., character set, viewport settings).
  • title: Document title.
Embedded Content
  • iframe: Inline frame.
  • audio: Audio content.
  • video: Video content.
Interactive Elements
  • button: Button.
  • details: Disclosure widget from which the user can obtain additional information or controls.

CSS

  • CSS stands for Cascading Style Sheets.
  • CSS is used for styling and layout of web pages.
Color and Background
  • color: Sets the text color.
  • background-color: Sets the background color.
  • background-image: Sets a background image.
Typography
  • font-family: Specifies the font for text.
  • font-size: Sets the size of the font.
  • font-weight: Sets the thickness of the font characters.
  • line-height: Sets the height of a line of text.
Layout
  • width and height: Sets the width and height of an element.
  • margin: Sets the margin outside the border of an element.
  • padding: Sets the padding inside the border of an element.
  • border: Sets the border properties of an element.
  • display: Specifies the type of box used for an HTML element.
  • position: Sets the positioning method of an element.
Flexbox and Grid
  • display: flex; and display: grid;: Enable Flexbox and Grid layout models for advanced layout control.
Alignment
  • text-align: Aligns text horizontally.
  • vertical-align: Aligns an inline or table-cell element vertically.
  • justify-content: Aligns items on the main axis in a flex container.
Visibility and Opacity
  • visibility: Specifies whether an element is visible or hidden.
  • opacity: Sets the transparency of an element.
Transformations
  • transform: Applies 2D or 3D transformations to an element.
Transitions and Animations
  • transition: Specifies the transition effect for an element.
  • animation: Defines keyframe animations.
Box Model
  • box-sizing: Determines how the sizing of an element's box model is calculated.

Git

  • git status: Checks what branch we are currently on.
  • git checkout -b branch-name: Creates a new branch and switches to it.
  • git add filename: Stages changes for commit.
  • git commit -m "Commit message": Commits changes to the repository.
Branching
  • git branch: Lists all branches in the repository.
  • git checkout branch-name: Switches to an existing branch.
  • git merge branch-name: Merges changes from one branch into another.
Remotes
  • git remote add origin remote-url: Adds a remote repository.
  • git push -u origin branch-name: Pushes changes to a remote branch.
  • git pull origin branch-name: Pulls changes from a remote branch.
Log and History
  • git log: Displays the commit history.
  • git diff: Shows changes between commits, branches, etc.
  • git blame filename: Shows who made changes to a file and when.
Undoing Changes
  • git reset commit-hash: Resets the staging area to a specific commit.
  • git revert commit-hash: Creates a new commit that undoes changes made in a previous commit.
  • git stash: Temporarily saves changes that are not ready to be committed.

JavaScript

  • JavaScript is a scripting language that enables interactive web pages.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
Variables and Data Types
  • let, const, var: Keywords for variable declaration.
  • String, Number, Boolean: Primitive data types.
  • Array: Ordered list of values.
  • Object: Collection of key-value pairs.
Control Flow
  • if, else if, else: Conditional statements.
  • switch: Selects one of many code blocks to be executed.
  • for, while: Loops for repetitive tasks.
Functions
  • function functionName(parameters) { /* code */ }: Defines a function.
  • return: Specifies the value a function should return.
  • Arrow functions: A concise way to write functions.
Objects and Classes
  • this: Refers to the current object in a method or constructor.
  • class ClassName { /* code */ }: Defines a class.
  • constructor: Method called when creating an instance of a class.
Asynchronous JavaScript
  • Callbacks, Promises, Async/Await: Handling asynchronous operations.
  • setTimeout, setInterval: Execute code after a delay or at intervals.
  • fetch API: Used to make network requests.
DOM Manipulation
  • document.getElementById, document.querySelector: Selecting DOM elements.
  • innerHTML, textContent: Updating element content.
  • addEventListener: Attaching event handlers to elements.
